<prism-js>

A Polymer element for syntax highlighting with Prism.js

Usage

Specify language & include code inline

<prism-js language="css"> p { color: blue; } </prism-js> Outputs: p { color: blue; }

Auto-escape HTML for you

<prism-js language="markup" escape="true">

Some paragraph text

 Outputs:

Some paragraph text

Attributes without value

<prism-js language="markup">

<p data-foo></p> <p data-foo ></p>

Name attribute pair

<prism-js language="markup">

<p class=prism></p>

Namespaces

<prism-js language="markup">

<html:p foo:bar="baz" foo:weee></html:p>

Tag that spans multiple lines

<prism-js language="markup">

<p >hello! </p>

Embedded JS and CSS

<prism-js language="markup">

<!DOCTYPE html> <html lang="en"> <head> <meta charset="utf-8" /> <title>I can haz embedded CSS and JS</title> <style> @media print { p { color: red !important; } } </style> </head> <body> <h1>I can haz embedded CSS and JS</h1> <script> if (true) { console.log('foo'); } </script> </body> </html>

LESS variables

<prism-js language="css">

@main-color: red; .foo { background: @main-color; }

Nested strings

<prism-js language="javascript">

var foo = "foo", bar = "He "said" 'hi'!"

Experimental features

There are currently limitations to how the polyfills handle stylesheet injection for ShadowDOM that limit how much we can dynamically work with injected styles.

Specify theme

<prism-js language="css" theme="okaidia"> p { color: blue; } </prism-js>

Include line-numbers

<prism-js language="javascript" linenumbers="true">

if (true) { console.log('foo'); }

Line Highlight

Single line: <prism-js language="css" linehighlight="5">

.line-highlight { position: absolute; left: 0; right: 0; padding: inherit 0; margin-top: 1em; /* Same as .prismâ€™s padding-top */ background: hsla(24, 20%, 50%,.08); background: -moz-linear-gradient(left, hsla(24, 20%, 50%,.1) 70%, hsla(24, 20%, 50%,0)); background: -webkit-linear-gradient(left, hsla(24, 20%, 50%,.1) 70%, hsla(24, 20%, 50%,0)); background: -o-linear-gradient(left, hsla(24, 20%, 50%,.1) 70%, hsla(24, 20%, 50%,0)); background: linear-gradient(left, hsla(24, 20%, 50%,.1) 70%, hsla(24, 20%, 50%,0)); pointer-events: none; line-height: inherit; white-space: pre; }

Combined with linenumbers: <prism-js language="css" linenumbers="true" linehighlight="3">

.line-highlight[data-end]:after { content: attr(data-end); top: auto; bottom: .4em; }

Ranges: <prism-js language="markup" linehighlight="3-8">

<!doctype html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Testing prism-js!</title> </head> <body> <h1>Yay!</h1> <script> var foo = 5; for (var i=0; i < foo; i++) { console.log('Iteration '+i); } </script> </body> </html>

Multiple lines: <prism-js language="markup" linehighlight="2, 11, 16">

<!doctype html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Testing prism-js!</title> </head> <body> <h1>Yay!</h1> <script> var foo = 5; for (var i=0; i < foo; i++) { console.log('Iteration '+i); } </script> </body> </html>

Multiple lines and ranges: <prism-js language="markup" linehighlight="2-3, 10-16">
